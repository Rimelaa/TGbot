Проект представляет собой Telegram бота, который помогает пользователям установить и отслеживать свои фитнес-цели. Бот предоставляет следующие функции:

1. Старт и помощь: Пользователь может начать общение с ботом, отправив команду /start. Бот приветствует пользователя и предоставляет список доступных команд, включая /help для получения справки.

2. Установка фитнес-цели: Пользователь может установить свою фитнес-цель, выбрав один из трех вариантов: набор мышечной массы, похудение или психологическая разгрузка. После выбора цели бот автоматически запрашивает у пользователя возраст, вес и рост.

3. Просмотр прогресса: Пользователь может просмотреть свой текущий прогресс, отправив команду /myprogress. Если пользователь уже установил свою фитнес-цель, бот отобразит ее, в противном случае предложит установить цель с помощью команды /setgoal.

4. Обработка сообщений: Бот обрабатывает текстовые сообщения пользователя, включая ответы на вопросы о цели, возрасте, весе и росте. Кроме того, он отвечает на любые другие текстовые сообщения, отправленные пользователем.

Проект использует библиотеку python-telegram-bot для работы с Telegram API и SQLAlchemy для взаимодействия с базой данных SQLite. Благодаря использованию ConversationHandler бот взаимодействует с пользователем последовательно, шаг за шагом, собирая необходимую информацию для установки фитнес-цели.
